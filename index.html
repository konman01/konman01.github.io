Web Based Secure Purchase Order system

Created the secure purchase order system using standard cryptographic algorithms and imitated the public key system

Designed and developed the security features for a purchase order system using cryptographic principles. We have applied security system where the payment transaction information will be encrypted and provided the data integrity for the data base at the server side. We also mimicked the public key infrastructure using java socket programming.


We have developed the purchase order system using the Servlet and JDBC apis, and mysql for backend database to store the user and payment related information. The service for public cryptographic system is designed using socket programming.

  • Date: November 2018
  • Category: Academic Project

Denial of service attack and Page Rank algorithm

Implementation of Master and Slave bots for generating denial of service attacks and Rise and fall of pagerank algorithm

Implemented Master and slave infrastructure where all the slave bots perform operations such as connecting and disconnecting to a specific server, getting the response page from a server and keeping the connection alive with the servers connected for a long time, on the command from the master bot. Also we have created Rise and fall of page rank algorithm using the mentioned master and slave architecture using Java Socket programming.
